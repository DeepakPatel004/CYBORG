A highly interactive, production-ready landing page featuring a dark-purple cybernetic aesthetic. Developed as **Task 1 (Web Development)** for the Techfest IIT Bombay College Ambassador Program. 

This project demonstrates advanced frontend capabilities, focusing on custom CSS animations, DOM manipulation, and immersive user micro-interactions without the use of external animation libraries.

## 🔗 Live Demo
**[View the Live System Here](https://deepakpatel004.github.io/CYBORG/)**

## ✨ Core Features

* **System Boot Sequence:** A simulated, full-screen terminal preloader that executes before revealing the main interface.
* **Magnetic UI Elements:** Call-to-action buttons feature a custom JavaScript magnetic pull effect and intense purple neon hover states.
* **3D Mouse-Tracking Parallax:** Feature cards and media elements track cursor coordinates (`offsetX`, `offsetY`) to calculate dynamic `rotateX` and `rotateY` CSS transforms.
* **Dynamic Diagnostics Dashboard:** Real-time simulated hardware metrics (Core Temp, Neural Sync, Latency) with animated CSS progress bars updating via JavaScript intervals.
* **Interactive Terminal Console:** A functioning mini-console allowing users to type basic commands (e.g., `help`, `clear`) for simulated system responses.
* **Flawless Responsiveness:** Engineered with CSS Grid and Flexbox to ensure a seamless experience from mobile devices to ultra-wide desktop monitors.

## 🛠️ Technical Stack

* **Structure:** HTML5 (Semantic framing)
* **Styling:** CSS3 (Custom properties, advanced keyframes, clip-paths, backdrop-filters, responsive grid/flex)
* **Logic:** Vanilla JavaScript (Intersection Observers, Event Listeners, DOM manipulation)
* **Typography:** `Orbitron` & `Rajdhani` (Google Fonts)
